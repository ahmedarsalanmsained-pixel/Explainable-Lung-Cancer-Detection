# Explainable-Lung-Cancer-Detection
An Explainable Hybrid Deep Learning Framework for Lung Cancer Detection from CT Images using U-Net, DCGAN, EfficientNetV2, CBAM, and Grad-CAM.
---

## 📖 Overview

This repository contains the implementation of my Master's research project, which proposes an explainable hybrid deep learning framework for lung cancer detection from CT scan images.

The framework integrates CLAHE for image enhancement, U-Net for lung segmentation, DCGAN for synthetic data augmentation, EfficientNetV2 for feature extraction, CBAM for attention-guided learning, and Grad-CAM for explainable AI.


 Key Features

- CLAHE Image Enhancement
- U-Net Lung Segmentation
- DCGAN Data Augmentation
- EfficientNetV2 Transfer Learning
- CBAM Attention Module
- Grad-CAM Explainable AI
- Multi-Class Lung Cancer Classification


Dataset

**Dataset:** IQ-OTH/NCCD Lung Cancer CT Dataset

Classes:

- Benign
- Normal
- Malignant


Results

- Classification Accuracy: **99.09%**
- Average ROC-AUC: **0.9999**
- 100% Recall for Malignant Cases
